# Walmart Sales Intelligence Dashboard

## 🚀 Overview
Developed an interactive analytics dashboard to monitor Walmart store performance, identify trends, and forecast future sales.

## ⚡ Features
- KPI metrics (Total, Avg, Max, Min)
- Sales trend visualization
- Forecasting using Linear Regression
- Sales drop alerts (anomaly detection)
- Store comparison analysis
- Growth trend tracking

## 🧠 Tech Stack
Python, Pandas, NumPy, Plotly, Streamlit, Scikit-learn

## 📊 Business Impact
- Identifies top-performing stores
- Detects sales anomalies early
- Enables data-driven decision-making
